## 面向对象

面向对象主要要学的是三大特性

* 封装

* 继承

* 多态

类定义

类中有属性和方法
类创建使用

``` java
class ClassName{

  ​	属性;

  ​	方法;

}
```

栈 堆

``` java
栈是用来存放变量的空间
堆是用来存放真正数据的地方

new 就会开辟一块堆空间
```
